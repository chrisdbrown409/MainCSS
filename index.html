Cascading Style Sheets

Cascading Style Sheets, or CSS works in conjunction with HTML. Consider the HTML code the framework of a website while CSS defines the exterior or the appearance of the website.

CSS can be added at the site location, page location or for a specefic line (in-line). CSS can be leveraged to define different attritubes such as fonts or padding (spaces). CSS is also used to control how websites are viewed on different devices.
